# Magic UI Startup Template

A premium, high-performance startup landing page template built with **Next.js**, **Tailwind CSS**, **Framer Motion**, and **Magic UI**.

## üöÄ Features

- **Modern Tech Stack**: Next.js 15+, TypeScript, and Tailwind CSS.
- **Magic UI Components**: Stunning animations using `BorderBeam`, `Marquee`, `Particles`, and more.
- **Responsive Marketing Pages**:
  - [x] **Landing Page**: Hero, Client logos, Pricing, and CTA sections.
  - [x] **Pricing**: Flexible pricing tables with monthly/yearly toggles.
  - [x] **FAQ**: Clean, accessible frequently asked questions.
  - [x] **Email Collection**: Dedicated waitlist/newsletter signup page.
  - [x] **Legal**: Professional Terms of Service and Privacy Policy pages.
- **Authentication**: Pre-built layouts for Login and Sign-up.
- **Theme Support**: Built-in dark mode and light mode using `next-themes`.
- **Form Handling**: Integrated with `react-hook-form` and `zod` for validation.

## üèÅ Getting Started

### Prerequisites

- Node.js 18+ installed.
- pnpm, npm, or yarn.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/abdelghanix10/Startup-Template-Magic-UI.git
   cd Startup-Template-Magic-UI
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Start the development server:

   ```bash
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## üìÅ Project Structure

- `app/`: Next.js App Router pages and layouts.
- `components/`:
  - `landing/`: Sections specific to the landing page.
  - `magicui/`: Animated components powered by Magic UI.
  - `ui/`: Common UI components (buttons, inputs, etc.).
- `lib/`: Utility functions and shared logic.
- `public/`: Static assets and images.
